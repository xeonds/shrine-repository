## 特性

作为一个不怎么喜欢数据库的人，我果断没有用任何数据库。我转向了markdown，并借鉴了hexo的魔改md思想（雾），把文章信息、文章内容、评论区都塞到了一个markdown文件里。
* 无数据库，开箱即用
* 支持markdown（不过是类似于hexo那样魔改的格式）
* 在线编辑(很简陋，后面完善下)
* 文章管理、数据统计（总字数，文章总数、标签数。后面添加更多信息）
* 文章评论（数据以特殊格式保存在该文章尾部）
